# final_package.zip manifest template

The final package should include:

- `README.md`
- `requirements.txt`
- `src/`
- `scripts/`
- `data/samples/`
- `data/summary/`
- `results/dataset_validation/`
- `results/pooling_designs/`
- `results/method_outputs/`
- `results/comparison/`
- `results/figures/`

Do not include temporary caches:

- `__pycache__/`
- `*.pyc`
- `.DS_Store`
- temporary staging directories under `outputs/`
