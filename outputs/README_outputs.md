# outputs directory

This directory stores the final deliverable ZIP created by the pipeline.

Generated by:

```bash
python scripts/run_pipeline.py
```

Expected final file:

```text
outputs/final_package.zip
```

`final_package.zip` should contain the GitHub-uploadable project files, including:

```text
README.md
requirements.txt
src/
scripts/
data/samples/
data/summary/
results/
```

At this stage, this directory is only prepared as the output destination. The final package ZIP should be created after the complete pipeline has run.
