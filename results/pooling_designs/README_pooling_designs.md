# pooling_designs

This directory stores pooling matrices and pool membership lists produced by Baseline 3 and the Proposed Method.

Generated by:

```bash
python scripts/03_run_all_methods.py
```

Expected layout:

```text
results/pooling_designs/
├── sample_001/
│   ├── baseline3_random_A.npy
│   ├── baseline3_random_pools.csv
│   ├── baseline3_random_info.json
│   ├── proposed_A.npy
│   ├── proposed_pools.csv
│   └── proposed_info.json
├── sample_002/
└── ...
```

Baseline 3 uses `build_random_pooling_matrix()`.
The Proposed Method uses `build_proposed_pooling_matrix()` with 4 groups and snake placement.
