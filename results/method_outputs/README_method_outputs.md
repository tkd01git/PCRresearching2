# method_outputs

This directory stores raw result CSVs from the four methods.

Generated by:

```bash
python scripts/03_run_all_methods.py
```

Expected files:

```text
baseline1_multistage.csv
baseline2_risk_ranking_only.csv
baseline3_random_sparse.csv
proposed_method.csv
all_methods_raw.csv
```

Required common columns:

```text
sample_id
method
pool_size
true_positive_count
asymptomatic_positive_count
symptomatic_positive_count
initial_pool_count
positive_pool_count
candidate_count
individual_tests
total_tests
detected_count
recall
status
```

Important rule: all rows must be computed from the same canonical `data/samples/sample_XXX/` directories.
