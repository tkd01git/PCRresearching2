# dataset_validation

This directory contains validation outputs for the canonical 50 samples.

Generated by:

```bash
python scripts/02_validate_datasets.py
```

Expected files:

```text
dataset_summary.csv
dataset_validation_report.md
```

Validation policy for the current dataset:

- The current canonical samples are copied from the existing generated `samples_asymptomatic_2of3` data.
- Symptoms are not re-normalized.
- The observed mean asymptomatic-positive rate is treated as about 0.60.
- The validation reference is `EXPECTED_ASYMPTOMATIC_POSITIVE_RATE = 0.60` with tolerance `0.15`.
